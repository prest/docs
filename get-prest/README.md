# Get pREST

There are several ways of downloading pREST, some are listed below. If it is your first time, we recommend following [Deploying with Docker](../deployment/deploying-with-docker.md).

* Use the Docker Hub image
* Use the GitHub image
* Download from [source](start-with-golang.md) and run
* Download the latest stable v1 binary from [releases/latest](https://github.com/prest/prest/releases/latest)

## v2.0.0

**v2.0.0** is the latest stable v2 release. It includes multi-database support, config resilience, JWT auto-disable, OR clause filtering, structured JSON logging, and per-user permissions. See the [release notes](../releases/v2.0.0.md) and [upgrade guide](../get-started/upgrading-to-v2.md).

* **Binary:** [v2.0.0 assets](https://github.com/prest/prest/releases/tag/v2.0.0)
* **Docker:** `prest/prest:v2.0.0`
